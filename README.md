## 🎯 项目动机：为什么我们学了 12 年数学却讨厌数学？

中学数学老师最常见的一句话是：
> "你以后买菜会用求根公式吗？"

这是数学教育最惨烈的一次自我背叛。它把**人类两千年最壮丽的智识冒险**，矮化成了"加减乘除买菜论"。

真实的数学史不是这样的：

- **二次方程**的求根公式刻在公元前 1800 年的巴比伦泥板上 —— 那是为了**分配土地**和**计算粮仓**。
- **三次方程**的解法引发了 16 世纪意大利数学家之间的**公开决斗**（Cardano vs Tartaglia），赌注是名誉和饭碗。
- **五次方程没有根式解** —— 这个结论让 20 岁的阿贝尔贫病而死，让 21 岁的伽罗瓦在决斗前夜写下了**群论**，开启了整个现代代数。
- **微积分**不是牛顿一个人在苹果树下发明的 —— 它是**两千年的接力赛**：阿基米德的穷竭法、印度喀拉拉学派的级数、开普勒算酒桶体积、费马求切线、巴罗（牛顿的老师）证出微积分基本定理的几何雏形……牛顿和莱布尼茨只是在 1666 年前后**同时收尾**，接着两人为了优先权**互相指控剽窃了三十年**，把英国数学和欧陆数学撕裂了一个世纪。
- **傅里叶级数**诞生于 1807 年，是为了解决**蒸汽机金属部件的热传导**问题 —— 直接服务于工业革命。
- **矩阵和线性代数**也是**跨越两千年的合作**：中国《九章算术》在公元前就用消元法解五元一次方程组（比高斯早 1500 年），日本关孝和（1683）比莱布尼茨更早写下行列式，高斯为了**找回失踪的小行星谷神星**发明了最小二乘法，凯莱和西尔维斯特在 1850 年代才把"矩阵"作为一种代数对象正式命名 —— 而这一切，最后成了量子力学、Google PageRank 和深度学习的共同语言。

马斯克在清华的演讲里说过类似的话：物理教育的失败，是没有教学生**第一性原理**和**真实世界的物理意义**，只让他们背公式应付考试。数学也一样。

**这个项目的目标**：用代码和动画，把数学还原成它本来的样子 —— 一部**带血带肉的人类思想史**。

### Chapter 0 · 视角即解法：为什么"换个角度看，难题瞬间消失"

> "数学家不解决问题，他们改写问题。" —— 笛卡尔式的精神

1787 年，9 岁的高斯被老师罚算 $1+2+\cdots+100$。他几秒钟交卷：5050。
他不是算得快，而是**看到了别人看不到的对称性**：

```
  1 +   2 +   3 + ... +  98 +  99 + 100
100 +  99 +  98 + ... +   3 +   2 +   1
─────────────────────────────────────────
101 + 101 + 101 + ... + 101 + 101 + 101    =  100 × 101 = 10100,  ÷2 = 5050
```

这不是"小聪明"，而是**视角的革命** —— 他把"加法"看成了"配对的对称"。
**几乎所有被罗列在教辅书上的"解题技巧"，本质上都是同一件事：换一个表征，让原来藏起来的结构暴露出来。** 难和易的差异，不在题目本身，而在**你戴的那副结构眼镜**。

#### 0.1 历史上反复出现的"换视角"母题

| 技巧 | 它在偷偷利用的结构 | 历史源头 | 现代化身 |
|------|------------------|---------|---------|

| **消元法**（《九章算术》前 200） | 线性方程组的行空间 | 刘徽注、高斯 1810 | LU 分解、Gauss–Jordan |
| **裂项相消**（莱布尼茨 1672） | 离散版"微积分基本定理"  $\sum (a_{n+1}-a_n) = a_N - a_0$ | 数列差分演算 | Stokes 定理、上同调 |
| **合并同类项** | 商环 / 等价类 | 花拉子米 *al-Jabr* (820) | 群环、张量收缩 |
| **配方法**（巴比伦 前 1800） | 二次型的正交对角化 | 巴比伦泥板 BM 13901 | 主成分分析、SDP |
| **换元法**（费马 1630s、欧拉 1750s） | 坐标变换 / 流形局部图 | 笛卡尔解析几何 | 微分同胚、归一化流 |

| **生成函数**（欧拉 1740） | 把序列编码成解析函数 | 《无穷分析引论》 | $z$ 变换、母函数法 |
| **对偶变换**（庞色列 1822） | 点 ↔ 线的射影对称 | 射影几何 | 凸优化对偶、傅里叶变换 |
| **不变量法**（克莱因 1872） | 群作用下的不变量 | 埃尔朗根纲领 | 李群、规范场论 |

#### 0.2 为什么是高斯"一下知道"？—— 直觉是历史的浓缩

把"高斯一下看出 $1+\cdots+n=\frac{n(n+1)}{2}$"归功于天赋，是教育学上最便宜的偷懒。事实是：

1. **对称即简化**这条母题，从巴比伦人切土地、毕达哥拉斯排石子（"三角形数"）开始，就在数学传统中流淌。高斯只是**继承了这条直觉**。
2. 教辅书把这些技巧**罗列**成几十个独立的招式，掩盖了它们其实都是同一件事的不同投影。学生背招式 → 永远只是"做过的题会、没做过的不会"；看清结构 → "没见过的题也能换个角度试试看"。
3. 这条直觉**可以被训练**：每当你遇到一个能"换视角变简单"的瞬间，就在脑里给它打上标签 —— **是哪种对称？哪种不变量？哪种同构？** 几年下来，你也会"一下看出"。

> 一句话总结：**所有"灵感"的背后，都是某种对称性、不变性、或同构。**

**🔑 关键代码片段**（`fibonacci_spiral.py`：Vogel 的向日葵种子排列，黄金角 ≈ 137.5°）

```python
# 黄金角的弧度值 —— 大自然给出的"最不可整除"角度
golden_angle = math.pi * (3 - math.sqrt(5))
k = torch.arange(1, 801, dtype=torch.float)
sun_r     = torch.sqrt(k)              # 半径 ∝ √k：每颗种子占同样面积
sun_theta = k * golden_angle           # 角度按黄金角累加
sun_x = sun_r * torch.cos(sun_theta)
sun_y = sun_r * torch.sin(sun_theta)
```

`pascal_triangle.py`：帕斯卡三角的递推 + mod 2 自动浮现谢尔宾斯基分形

```python
T = torch.zeros(N, N, dtype=torch.long); T[0, 0] = 1
for i in range(1, N):
    for j in range(i + 1):
        T[i, j] = (T[i-1, j-1] if j > 0 else 0) + (T[i-1, j] if j < i else 0)
sierpinski = (T % 2).float()           # 一行 mod 2，1654 → 20 世纪的分形
```

### Chapter 2 · 从数列到微积分：两千年的接力赛
- **核心思想**：当你**让差分的间隔无限缩小**，离散的数列就变成了连续的曲线，**导数**和**积分**就诞生了。但这个"无限缩小"的念头，是**七十多代人**前赴后继才搞清楚的。

#### 2.1 史前史：从阿基米德到中世纪（公元前 250 → 公元 1600）
微积分不是从牛顿开始的。它有一段长达 **1900 年的"前传"**：
- **阿基米德（前 287–前 212，叙拉古）**：用"穷竭法"算出球的体积是外切圆柱的 $\frac{2}{3}$ —— 这个结论刻在他的墓碑上。他在罗马士兵攻城时仍在沙地上画图，被一剑刺死前最后一句是："别动我的圆。"他实际上已经做出了**积分**的雏形，但希腊数学没有"极限"的语言，这条路被冻结了 1800 年。
- **印度喀拉拉学派（1340–1500）**：马德哈瓦（Mādhava）算出了 $\sin$、$\cos$、$\arctan$ 的**无穷级数展开** —— 比牛顿和莱布尼茨早整整 **300 年**。但他们的成果用梵文写在棕榈叶上，没有传到欧洲，一直到 19 世纪才被重新发现。**数学的进步不是必然的，它需要传播**。
- **开普勒（1609，布拉格）**：为了精确描述火星轨道（用第谷·布拉赫积累 20 年的肉眼观测数据），他**算了 70 多遍**才发现行星轨道是椭圆。在《新天文学》里，他用"无穷小三角形"算椭圆面积 —— 这是现代积分的直接前身。他也是第一个用积分**算酒桶体积**的人（1615 年的《葡萄酒桶的新立体几何》），这本书的写作动机是他觉得酒商量酒桶的方法太离谱。

#### 2.2 17 世纪的爆发：通往牛顿的五条小径（1630 → 1666）
微积分的"最后一公里"，是六个人在三十年内**几乎同时**铺好的：

| 年份 | 人物 | 贡献 | 社会背景 |
|------|------|------|---------|
| 1635 | 卡瓦列里（意） | "不可分量法"：把面积看成无数线段之和 | 伽利略学生，被教会盯着 |
| 1636 | 费马（法） | "Adequality"求极值与切线 —— 导数的雏形 | 业余数学家，本职是律师 |
| 1637 | 笛卡尔（法） | 《几何学》：解析几何，把代数和几何打通 | 在《方法论》附录里发表 |
| 1656 | 沃利斯（英） | 无穷乘积、无穷小记号 $\infty$ 由他引入 | 政治内战中破解王党密码 |
| 1670 | 巴罗（英） | 几何形式的**微积分基本定理** | 牛顿的老师，后让位给牛顿 |

到 1660 年代，导数和积分的所有零件都已经在桌面上 —— **就差一个人把它们组装起来**。

#### 2.3 牛顿的"奇迹年"与莱布尼茨的独立发现（1665 → 1684）
- **1665–1666 大瘟疫**：剑桥关闭，23 岁的牛顿回到林肯郡乡下的伍尔索普庄园，**一个人在两年里**搞出了：流数法（微积分）、万有引力、光的色散三件大事。这就是科学史上著名的 *Annus Mirabilis* —— 但他**没发表**。他的笔记本上写满了 $\dot{x}$（流数 fluxion）和 $\dot{y}$，但他怕被批评，把手稿藏了 20 年。
- **1675，巴黎**：莱布尼茨为了制造**机械计算器**（人类第一台能做四则运算的机器）正在研究无穷级数。他**完全独立地**发现了微积分，并发明了我们今天还在用的记号 $\frac{dy}{dx}$ 和 $\int$。
- **1684**：莱布尼茨抢先发表论文。
- **1687**：牛顿在哈雷的催促下出版《自然哲学的数学原理》—— 但里面**故意不用流数法**，全部翻译成古希腊式的几何证明，因为他觉得"这样更高级"。

#### 2.4 优先权大战：撕裂欧洲数学一百年（1699 → 1716）
这是科学史上最丑陋的一章：
- 1699 年起，牛顿派系（英国皇家学会）公开指控莱布尼茨**抄袭**。
- 1712 年，皇家学会成立"中立调查委员会"裁决 —— 但**报告其实是牛顿本人秘密起草的**。
- 1716 年莱布尼茨在贫困中去世，葬礼只有他的秘书一人到场。
- **代价**：英国数学家出于民族情绪坚持用牛顿的笨拙记号 $\dot{x}$，而欧陆使用莱布尼茨更优雅的 $\frac{dx}{dt}$，结果**英国数学落后了整整一个世纪**，直到 1820 年代剑桥的"分析学会"运动才追上来。

#### 2.5 严密化危机：贝克莱主教的诘问（1734 → 1872）
微积分诞生时，**它的逻辑基础其实是错的**：
- **1734，贝克莱主教（爱尔兰）**：在小册子《分析学家》中嘲讽微积分里的"无穷小量" —— "它既不是有限量、也不是零、也不是无穷小，那它到底是什么？是**逝去量的幽灵**（the ghosts of departed quantities）。"这一击让数学家头疼了 100 年。
- **1797，拉格朗日**：试图把微积分纯代数化（避开极限），失败了。
- **1821，柯西（法）**：在巴黎综合理工学院讲义中，第一次用"极限"这个概念**精确**定义导数 —— 现代分析学的开端。但他自己有时也含糊其词。
- **1860s，魏尔斯特拉斯（柏林）**：发明 $\varepsilon$-$\delta$ 语言，**真正消灭了"无穷小幽灵"**。从牛顿到魏尔斯特拉斯，**整整两百年**，微积分才有了它应有的逻辑严密性。
- **1854，黎曼**：定义现代积分。**1902，勒贝格**：测度论与勒贝格积分，把积分推广到病态函数 —— 这是后来概率论和量子力学的基础。

#### 2.6 社会土壤：为什么是 17 世纪？
微积分不是凭空诞生的。它的出现需要三个条件**同时**成熟：
1. **天文学的精度危机**：哥白尼–开普勒–伽利略积累的行星观测数据（特别是第谷的火星记录），逼着数学家必须算"瞬时速度"和"瞬时位置"。
2. **军事与航海**：弹道学（炮弹轨迹）、经度问题（航海钟）、潮汐预测 —— 都是欧洲列强重金悬赏的实际问题。
3. **印刷术 + 科学院制度**：拉丁文论文可以快速在欧洲流通，1660 年的英国皇家学会、1666 年的法国科学院给了数学家**第一个职业舞台**。

### Chapter 4 · 线性代数：从《九章算术》到矩阵宇宙
- **核心思想**：当未知数从 2 个变成 100 个，"消元法"必须被**系统化** —— 矩阵不是高中老师说的"一堆数字"，而是**线性变换的语言**。但从"解方程组"到"矩阵作为代数对象"，人类走了 **2000 年**。

#### 4.1 古代：解方程组的实用智慧（前 200 → 公元 1000）
线性代数的源头不在欧洲，而在**东方**：
- **《九章算术·方程章》（约公元前 200 → 公元 263 刘徽注）**：第八章"方程"中明确给出了**五元一次方程组**的解法，使用"遍乘直除"算法 —— 这就是**高斯消元法**的完整原型，**比高斯早了整整 1500 年**。原文："今有上禾三秉，中禾二秉，下禾一秉，实三十九斗……"（这是一道土地分粮问题）。刘徽（263）作注时已经在用**负数**做消元，比欧洲早 1300 年（欧洲数学家在 16 世纪还把负数叫"荒谬数"）。
- **印度：婆罗摩笈多（628）、巴斯卡拉（1150）**：用一种叫 *kuṭṭaka*（"粉碎法"）的算法解线性丢番图方程 —— 服务于天文学（行星会合周期计算）。
- **波斯/阿拉伯：花拉子米（820）**：《代数学》（*al-Jabr*）系统化了一次和二次方程的解法 —— "代数"（algebra）这个词就来源于此。

#### 4.2 行列式的诞生：东西方独立发明（1683 → 1812）
"行列式"这个概念，在地球两端**几乎同时**被发明：
- **1683，关孝和（日本）**：在《解伏题之法》中给出了 **3 阶到 5 阶**行列式的展开公式 —— **比莱布尼茨早 10 年**。江户时代的"和算"流派由此达到巅峰。
- **1693，莱布尼茨**：在给洛必达的信里独立提出了行列式的概念，但没有发表。
- **1750，克莱姆（瑞士）**：发表"克莱姆法则" —— 用行列式解线性方程组的公式。
- **1772，范德蒙德（法）**：第一次把行列式作为**独立的研究对象**（不只是解方程的工具）。
- **1812，柯西**：把"determinant"（决定式）这个名字定下来，证明了行列式乘法定理。

#### 4.3 高斯与失踪的小行星：最小二乘法的真实故事（1801）
线性代数史上最戏剧性的一章：
- **1801 年 1 月 1 日**：意大利天文学家皮亚齐发现了第一颗小行星**谷神星**（Ceres），观测了 41 天后，谷神星运行到太阳背后**消失了**。整个欧洲天文界陷入恐慌：人类历史上第一颗发现的小行星，**就这样丢了**。
- **1801 年下半年**：24 岁的高斯接受挑战。他用一种当时还没公开的方法 —— **最小二乘法**（least squares）—— 从 41 天的稀疏观测数据反推轨道。
- **1801 年 12 月 31 日**：根据高斯预测的位置，天文学家**重新找到了谷神星**，误差极小。这一战让高斯名震欧洲。
- **1809**：高斯在《天体运动论》中公开最小二乘法。**勒让德 1805 年其实已经独立发表了这个方法**，于是又是一场（比较温和的）优先权之争。
- 最小二乘法的本质就是 $A^T A x = A^T b$ —— 这是今天**所有机器学习"训练"的最古老祖先**。

#### 4.4 矩阵作为代数对象：英国学派的发明（1850 → 1890）
"矩阵"这个词出现得很晚：
- **1850，西尔维斯特（英）**：第一次使用 "matrix"（矩阵）这个词 —— 拉丁语原意是"母体、子宫"，因为他认为矩阵是"生出行列式的母体"。
- **1858，凯莱**："A Memoir on the Theory of Matrices" —— 第一篇把**矩阵视为独立代数对象**的论文。定义了矩阵加法、乘法、逆矩阵、**凯莱-哈密顿定理**（每个矩阵满足自己的特征多项式）。
- **1878，弗罗贝尼乌斯（德）**：秩、相似变换、Jordan 标准形 —— 矩阵理论体系化。
- **社会背景**：19 世纪是**电报网络、铁路调度、保险精算**爆炸的时代，工程师需要系统化的方法处理大型线性方程组。

#### 4.5 向量空间的抽象化（1843 → 1888）
和矩阵平行，"向量"也在被发明：
- **1843，哈密顿（爱尔兰）**：在都柏林布鲁姆桥散步时，**当场刻在桥栏杆上**：$i^2 = j^2 = k^2 = ijk = -1$ —— 四元数诞生。"vector"（向量）一词由他引入。
- **1844，格拉斯曼（德，中学教师）**：《延伸论》（*Ausdehnungslehre*），独立建立了 $n$ 维向量空间和外代数 —— **完全超越他的时代**，被同行无视，他后来转去研究梵语，成了著名的语言学家。**直到 1880 年代他的数学才被重新发现**。
- **1880s，吉布斯 & 亥维赛**：从四元数中提炼出现代的向量分析（点积、叉积、$\nabla$ 算子）—— 麦克斯韦方程组的标准语言。
- **1888，皮亚诺**：第一次给出**抽象向量空间的公理化定义** —— 这是现代线性代数的起点。

#### 4.6 无穷维与量子革命（1900 → 1932）
线性代数走出"有限维"，迎来 20 世纪最壮丽的应用：
- **1900–1910，希尔伯特**：研究积分方程时引入"希尔伯特空间"—— 无穷维的内积空间。
- **1922，巴拿赫（波兰）**：泛函分析诞生，巴拿赫空间。
- **1925，海森堡**：在赫尔戈兰岛养花粉过敏，**重新发明了矩阵乘法**（他当时不知道矩阵是什么），写下"矩阵力学"。
- **1926，薛定谔**：波动力学（看似不同的另一种语言）。
- **1932，冯·诺伊曼**：《量子力学的数学基础》—— 证明矩阵力学和波动力学**本质上是同一个希尔伯特空间上的算子理论**。**线性代数在这一刻成为物理学的官方语言**。

#### 4.7 计算线性代数：从打孔卡到 GPU（1947 → 现在）
现代深度学习背后是一条隐秘的计算技术线：
- **1947，图灵**：分析数值线性方程组求解的舍入误差 —— 数值线性代数的开端。
- **1950s，Householder、Givens**：正交变换、QR 分解 —— 数值稳定的算法。
- **1965，Cooley-Tukey**：FFT 算法（快速傅里叶变换）—— 实质上是一个矩阵分解。
- **1979，LINPACK** → **1992，LAPACK** → **BLAS**：今天 NumPy / PyTorch / TensorFlow **底层调用的还是这些 1990 年代写的 Fortran 代码**。
- **2007，CUDA**：GPU 上的矩阵乘法 —— 深度学习革命的硬件基础。

> 一句话总结：**当你写下 `torch.matmul(A, B)` 时，你正在调用从《九章算术》刘徽、关孝和、高斯找谷神星、凯莱定义矩阵、海森堡的量子力学、到 1990 年代 LAPACK 工程师 —— 跨越 2200 年的智识遗产。**

**🔑 关键代码片段**（`bayes_disease_test.py` —— 99% 准确度的检验，对 1% 患病率的人群依然只有 ~50% 后验概率，这就是基率忽略）

```python
sens = 0.99   # P(阳性 | 患病)
spec = 0.99   # P(阴性 | 健康)

# 贝叶斯公式：P(患病|阳性) = sens·prev / (sens·prev + (1-spec)·(1-prev))
prevalences = torch.logspace(-5, -1, 200)
posterior = (sens * prevalences) / (
    sens * prevalences + (1 - spec) * (1 - prevalences)
)
# 当 prev = 1% 时，后验 ≈ 0.50 —— 一半的"阳性"其实是健康人
```

### Chapter 7 · 现代深度学习：站在所有巨人的肩膀上
- **核心思想**：当我们写下 `loss.backward()` 时，我们在同时使用：
  - 牛顿的**微积分**（求导）
  - 高斯的**线性代数**（矩阵乘法）
  - 拉格朗日的**最优化**（梯度下降）
  - 傅里叶的**频域分析**（卷积神经网络）
  - 贝叶斯的**概率论**（softmax / KL 散度）

**🔑 关键代码片段**（`mnist_from_scratch.py` —— 训练循环的 6 行代码，是 6 位历史人物的合作）

```python
# Cayley (1858)：一个全连接层 = 矩阵乘 + 偏置
W1 = (torch.randn(784, 128) * 0.05).requires_grad_(True)
b1 = torch.zeros(128, requires_grad=True)
W2 = (torch.randn(128, 10) * 0.05).requires_grad_(True)
b2 = torch.zeros(10, requires_grad=True)

def model(x):
    h = F.relu(x.view(x.shape[0], -1) @ W1 + b1)   # Cayley + Hahnloser (2000)
    return h @ W2 + b2

opt = torch.optim.SGD([W1, b1, W2, b2], lr=0.1)    # Cauchy 1847 + Robbins-Monro 1951

for x, y in train_loader:
    loss = F.cross_entropy(model(x), y)            # Boltzmann 熵 + Shannon 信息论
    opt.zero_grad()
    loss.backward()                                # Leibniz 链式法则，Rumelhart 1986 自动化
    opt.step()                                     # 牛顿流数 → x.grad
```

> **一行 `loss.backward()` 的背后**：牛顿（微分）+ 高斯（线性代数）+ 柯西（梯度下降）+ 玻尔兹曼（熵）+ 香农（交叉熵）+ 莱布尼茨（链式法则）—— **三百年数学史在 0.001 秒里同时复活。**
